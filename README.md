# ΔΜΘ Navigator

A digital directory and navigation tool for the **Thessaloniki Courthouse (Δικαστικό Μέγαρο Θεσσαλονίκης)**.

The project allows lawyers, citizens and visitors to quickly locate court services, offices and authorities operating inside the courthouse.

Instead of searching through physical directories or asking around the building, users can instantly find the correct office, floor and contact information.

## Features

### Smart Search
Instant search across all courthouse services.

### Court Services Directory
Structured listing of authorities and offices operating inside the courthouse.

### Mobile Friendly
Fully responsive interface optimized for smartphones.

### Lightweight
Built using vanilla web technologies without heavy frameworks.

### Click-to-Call
Phone numbers are clickable for mobile users.

### Floor Information
Each entry displays the floor location inside the building.

---

## Project Purpose

Courthouses often contain many services and offices, which can be difficult to locate for both professionals and citizens.

ΔΜΘ Navigator aims to provide:

- clear navigation  
- fast search  
- accessible information  

through a simple digital interface.

---

## Legal Tech Context

This project demonstrates how legal informatics and digital tools can improve everyday workflows inside judicial institutions.

It serves as an example of how small-scale technology solutions can modernize legal infrastructure without complex or expensive systems.
