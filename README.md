🏛 ΔΜΘ Navigator

A digital directory and navigation tool for the Thessaloniki Courthouse (Δικαστικό Μέγαρο Θεσσαλονίκης).

The project helps lawyers, citizens, and visitors quickly locate court services, offices, and public authorities inside the courthouse.

Instead of manually searching through floor lists or asking around the building, users can instantly search and discover the correct office, floor, and contact information.

🔗 Live website:
https://jesuisperdu3301.github.io/dmth-navigator/

✨ Features

🔎 Smart Search
Search for services, offices, or organizations instantly.

🏛 Court Directory
Structured listing of services operating inside the courthouse.

📱 Mobile Friendly
Optimized interface for smartphones and tablets.

⚡ Fast & Lightweight
Built with simple web technologies — no heavy frameworks.

🧭 Floor Navigation
Each entry displays the floor location inside the courthouse.

📞 Direct Contact
Telephone numbers are clickable on mobile devices.

🧱 Tech Stack

The project is intentionally simple and transparent.

No frameworks or backend were required, making the project extremely lightweight and maintainable.
